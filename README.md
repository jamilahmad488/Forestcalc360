# ForestCalc 360

**ForestCalc 360** is an AI-assisted forestry calculator for field and desktop use.  
It allows foresters, community forestry teams, and carbon project developers to calculate:

- Standing Tree Volume (Chir Pine, Blue Pine, Deodar, Fir/Spruce, Oak, Wild Olive)
- Log Volume (Smalian, Huber, Newton formulas)
- Scants / Timber volume (CFT)
- Biomass and Carbon Sequestration (tCO2e)

---

## Features

- Simple, user-friendly interface
- Dropdown menu for species selection
- Auto-height estimation using DBH if height not measured
- Volume per tree & total stand volume
- Mobile-friendly interface
- Exportable results (Excel/PDF)
